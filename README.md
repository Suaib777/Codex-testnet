# Codex Testnet

A lightweight Python starter project for demonstrating structured commits, documentation, and testing conventions.

## Highlights
- Simple Python module with a CLI entry point.
- Minimal test suite using `pytest`.
- Conventional project scaffolding (.gitignore, editorconfig, etc.).

## Quick start
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m codex_testnet --help
```

## Usage
```bash
python -m codex_testnet "Codex"
```

## Common tasks
```bash
make install
make test
```
